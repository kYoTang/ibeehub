A plugin that helps in developing premium themes

== Description ==

This plugin helps in developing premium themes. It turns free themes at wp.org into Pro themes.

== Installation ==

You won't need to install this separate as it is required by actual Pro theme plugin.
This was required plugin and installed by TGMPA script. Anyways, you can install this
as you would do any other plugin.

1. Download and unzip 'webulous-framework.zip' file.
2. Upload `webulous-framework` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
